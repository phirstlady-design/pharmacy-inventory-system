# Pharmacy Inventory System

A web-based Pharmacy Inventory Management System developed to help pharmacies manage medicines, sales, stock levels, suppliers, and expiry tracking efficiently.

## Features

- User Authentication
- Dashboard Analytics
- Medicine Management
- Inventory Tracking
- Stock In / Stock Out
- Sales Management
- Supplier Management
- Expiry Alerts
- Reporting

## Technology Stack

- PHP
- MySQL
- JavaScript
- Bootstrap
- HTML/CSS

## Installation

1. Clone repository

```bash
git clone https://github.com/phirstlady-design/pharmacy-inventory-system.git
```

2. Import database

```text
pharmacy_inventory.sql
```

3. Configure database connection

```php
$conn = mysqli_connect(
    'localhost',
    'your_db_user',
    'your_db_password',
    'your_database'
);
```

4. Run using XAMPP
